전 세계의 친구들과 회원 가입 없이 익명으로 
랜덤 형식의 쪽지 보내기 기능을 제공합니다.  (사진,음성도 첨부 가능)
사용 언어가 다른 상대와는 주고 받은 메시지 번역 기능 제공


[개요] 
전 세계의 친구들과 회원 가입 없이 익명으로 
랜덤 형식의 쪽지 보내기 기능을 제공합니다.  (사진,음성도 첨부 가능)
사용 언어가 다른 상대와는 주고 받은 메시지 번역 기능 제공


[주요기능] 
- 회원 가입 및 인증 처리 없음 순수 익명 서비스
- 익명의 친구들에게 메세지 보내기
- 음성 메시지 보내기 최대(1분) 
- 사진 메시지 보내기 
- 화면 잠금 기능 제공
- 단말 기기의 어떠한 정보도 요구하거나 서버에 저장 하지 않음 
- 결재 관련 정보 요청 없음

[지원OS]
- 안드로이드 5.0 LOLLIPOP 이상 

[지원해상도]
- 1280 * 720 해상도 비율에 최적화(1920x1280, 2560x1440) 
- 저 해상도의 단말도 일부 글자나 화면이 크게 보일 수 있지만
  모든 기능을 지원 합니다. 
  
[지원디바이스]
- 안드로이드 5.0 LOLLIPOP 이상 OS를 사용하는 모든 디바이스를 지원합니다. 

[접근 권한]
권한은 해당 기능 사용 시 동의가 필요하며,허용하지 않아도 
해당 기능 외 서비스 이용이 가능합니다.

□ 필수 권한
- 필수 권한 항목 없음

- 음성   : 음성 녹음
